# Hangman

Hangman is a game where you have to guess the word

## Installation

```bash
#move to working directory
cd src/main/java 

#compile .class file
javac Main.java

#build .jar file
jar cvfe Hangman.jar  Main *.class 

#execute .jar file
java -jar Hangman.jar 
```

Hangman.jar can be executed anywhere
